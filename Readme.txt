This will be used as the repository for all code and assets. 
The main branch will be created and each user shall create a branch that forks from the main branch.
If any changes to the code are to be made, they shall be done on the user branch and a pull request shall be initiated and after a code review, shall be merged into the main branch.

Branches shall be named in the following manner to keep track of work:
Sprint Iteration/Name/Feature
